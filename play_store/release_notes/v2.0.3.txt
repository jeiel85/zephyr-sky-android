v2.0.3 Live weather API integration

Added
• Real weather, hourly forecast, and air-quality powered by the free Open-Meteo API (no API key required)
• City lookup via Open-Meteo Geocoding for latitude/longitude resolution
• INTERNET and ACCESS_NETWORK_STATE permissions

Changed
• Removed hard-coded mock data
• Sunrise/sunset times now come from live API responses
• WMO weather codes mapped to the in-app conditions (clear/cloudy/rain/snow/thunder)
</en-US>
