## v2.0.6 - Zephyr Sky 브랜드 잔여 표기 정리

### 주요 변경 사항
- 앱 상단 헤더에 남아 있던 이전 브랜드명 `Skyline Weather`를 `Zephyr Sky`로 정리했습니다.
- 보조 태그라인을 현재 앱 성격에 맞는 미니멀 날씨 앱 문구로 변경했습니다.
- 테스트 날씨 경보 알림 제목에 남아 있던 `skyline` 표기를 `Zephyr Sky`로 변경했습니다.

### 수정 사항
- 내부 설정 저장소 이름을 `zephyr_sky_prefs`로 전환했습니다.
- 기존 `skyline_weather_prefs`에 저장된 마지막 도시/위치, 테마, 온도 단위, 알림 설정은 새 저장소로 복사해 유지되도록 했습니다.

### 문서 / 빌드 / 배포
- 배포 문서와 Play Store 문서의 예전 `zephyr-sky` Pages URL을 현재 `zephyr-sky-android` URL로 정렬했습니다.
- ROADMAP의 이전 에디토리얼 저널/Gemini/CameraX 중심 설명을 현재 Zephyr Sky 미니멀 날씨 앱 기준으로 갱신했습니다.
- 앱 내부 버전을 `versionName 2.0.6`, `versionCode 206`으로 올렸습니다.

### 검증
- 로컬: `test`, `lint`, `assembleDebug`, `bundleRelease` 성공.
- CI: GitHub Actions Android Build & Release에서 테스트, APK/AAB 빌드, GitHub Release 생성을 확인합니다.
- 산출물: 로컬 Play Store 제출용 `zephyr-sky-v2.0.6.aab` 및 `zephyr-sky-v2.0.6-release-notes.txt` export 확인.

### 설치 또는 업데이트 참고 사항
- 기존 사용자의 도시, 위치, 테마, 단위, 알림 설정은 새 설정 저장소로 자동 복사됩니다.
