## v2.0.1 — Android Native 리뉴얼 후속 정리

### 주요 변경 사항
- Flutter 기반 구버전에서 Kotlin/Jetpack Compose Android Native 구조로 전환한 뒤 남아 있던 빌드와 배포 흐름을 Gradle 기준으로 정리했습니다.
- 앱 런처 및 Activity 표시명이 템플릿 값 `Weather`로 보이던 문제를 `Zephyr Sky`로 바로잡았습니다.
- GitHub Issues에 남아 있던 리뉴얼 이전 백로그 29건을 닫고, 리뉴얼 이후 기준으로 새 작업을 시작할 수 있도록 초기화했습니다.

### 문서 / 빌드 / 배포
- `DEPLOYMENT.md`를 Android Native 빌드, GitHub Actions, 릴리즈 서명, APK/AAB 산출물 기준으로 갱신했습니다.
- 릴리즈 전에 수기 GitHub Release 노트를 작성하는 `docs/releases/` 체계를 추가했습니다.
- Play Console에 붙여넣을 다국어 릴리즈 노트를 `play_store/release_notes/`에 별도로 관리하도록 추가했습니다.
- 태그 릴리즈 워크플로우가 `docs/releases/vX.Y.Z.md`를 GitHub Release 본문으로 사용하도록 변경했습니다.

### 설치 또는 업데이트 참고 사항
- 이번 버전은 Android Native 리뉴얼 직후의 정리 릴리즈입니다.
