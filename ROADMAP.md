# Zephyr Sky - 에디토리얼 날씨 저널 개발 로드맵

## 프로젝트 개요
**목표:** 단순한 기상 예보 정보 표시를 넘어, 현재 위치의 실시간 날씨 데이터와 사용자의 감정을 기반으로 인공지능(Gemini API)이 한 편의 아름다운 '에디토리얼 날씨 저널'을 자동으로 생성 및 작성을 돕고, 순간의 사진을 캡처해 기록으로 보존하는 차세대 감성 날씨 저널 모바일 앱 개발.

## 🗺️ 개발 로드맵 (5단계)

### Phase 1: Gemini AI 연동 및 날씨 파싱 (완료)
* **목표**: 날씨 데이터를 텍스트 프롬프트로 변환하여 Firebase AI SDK (Gemini)에 전달하고, 독창적인 기상 에디토리얼 저널을 출력받아 사용자에게 제시합니다.
* **구현 세부**:
  - `WeatherApi.kt`, `WeatherModels.kt`로 REST 기상 정보 처리.
  - Gemini API 연동을 통한 에디토리얼 템플릿 실시간 작문.

### Phase 2: 로컬 영속 스토리지 및 저널 내역 보존 (완료)
* **목표**: 사용자가 작성하고 편집한 날씨 저널을 로컬에 영구히 저장하고, 과거 작성 내역을 날짜별/날씨별로 필터링해 볼 수 있는 아카이브 환경을 구성합니다.
* **구현 세부**:
  - Room Database를 도입하여 테이블 설계 및 이관 지원.
  - DataStore Preferences를 이용한 단위 및 마지막 위치 저장 설정 관리.

### Phase 3: CameraX 모바일 카메라 통합 (완료)
* **목표**: 사용자가 저널 작성 시 오늘의 하늘이나 주변 분위기를 즉석 촬영하고 기록에 첨부해 보관할 수 있도록 모바일 카메라 환경을 내장합니다.
* **구현 세부**:
  - Android Jetpack CameraX 모듈을 이용한 프리뷰 구현.
  - 이미지 캡처 후 내부 로컬 스토리지에 URI로 저장 및 Room DB 연결.

### Phase 4: 스크린샷 기반 비주얼 회귀 테스트 구축 (완료)
* **목표**: UI 컴포넌트의 깨짐 현상이나 디자인 회귀를 방지하기 위해 로컬 단위 테스트 수준에서 UI 화면을 렌더링하고 이미지를 대조 검증하는 환경을 구축합니다.
* **구현 세부**:
  - Robolectric 기반 가상 디바이스 테스트 환경 설정.
  - Roborazzi 플러그인을 활성화하여 `app/src/test/screenshots/` 내 스크린샷 생성 및 대조 테스트 실행 가능 확보.

### Phase 5: 다국어 고도화 및 위젯 지원 (예정)
* **목표**: 에디토리얼 저널의 다국어(한국어, 영어 등) 생성 프롬프트를 정교화하고, 안드로이드 홈 스크린에 오늘의 저널 스니펫과 실시간 날씨 요약을 보여주는 위젯을 개발합니다.
* **구현 세부**:
  - 다국어 번들 리소스 리팩토링.
  - Android Glance 또는 표준 AppWidget 라이브러리를 활용한 홈 위젯 연동.
