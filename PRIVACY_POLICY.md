# 개인정보 처리방침

**Zephyr Sky (제퍼 스카이)**
**최종 수정일: 2026년 5월 26일**

이 문서는 Zephyr Sky Android 앱의 현재 빌드 기준 데이터 처리 방식을 설명합니다.

## 1. 소개

Zephyr Sky는 광고와 추적 없이 날씨 정보를 제공하는 Android 앱입니다. 본 앱은 사용자가 입력한 도시명, 사용자가 직접 선택한 현재 위치, 앱 설정을 필요한 범위에서만 처리하며, 개인을 직접 식별하는 이름, 이메일, 전화번호를 수집하지 않습니다.

## 2. 처리하는 정보

### 2.1 도시 검색어

- 처리 항목: 사용자가 검색창에 입력한 도시명.
- 사용 목적: 해당 도시의 좌표를 찾고 날씨, 시간별 예보, 대기질 정보를 조회하기 위함.
- 보관 방식: 마지막으로 선택한 도시는 기기 내부 설정에 저장될 수 있습니다.

### 2.2 앱 설정

- 처리 항목: 마지막 도시 또는 위치 좌표, 섭씨/화씨 선택, 테마 설정, 알림 사용 여부.
- 사용 목적: 앱을 다시 열었을 때 사용자가 선택한 상태를 유지하기 위함.
- 보관 방식: Android SharedPreferences를 통해 기기 내부에만 저장됩니다.

### 2.3 위치 권한

- 처리 항목: 사용자가 현재 위치 버튼을 누른 경우 기기에서 제공하는 대략적인 위치 좌표.
- 사용 목적: 현재 위치의 날씨, 시간별 예보, 대기질 정보를 조회하기 위함.
- 위치 권한은 선택 사항이며, 기기 설정에서 언제든지 철회할 수 있습니다.

### 2.4 알림 권한

- 처리 항목: Android 알림 권한 허용 여부.
- 사용 목적: 현재 기온 알림과 날씨 경보 알림을 표시하기 위함.
- 알림 권한은 선택 사항이며, 기기 설정에서 언제든지 철회할 수 있습니다.

> 현재 앱은 대략적인 위치 권한만 선택적으로 요청하며, 카메라, 연락처, 마이크, 광고 ID를 사용하지 않습니다.

## 3. 외부 서비스 제공

Zephyr Sky는 날씨 정보 제공을 위해 Open-Meteo 서비스를 사용합니다.

- Geocoding API: 사용자가 입력한 도시명을 좌표로 변환합니다.
- Forecast API: 좌표를 기준으로 현재 날씨와 시간별 예보를 조회합니다.
- Air Quality API: 좌표를 기준으로 대기질 정보를 조회합니다.

외부 통신은 HTTPS로 이루어지며, 앱은 사용자 식별 정보나 광고 식별자를 함께 전송하지 않습니다.

## 4. 광고 및 분석 도구

Zephyr Sky는 광고 SDK, Firebase Analytics, Crashlytics 등 사용자 행동 분석 또는 추적 도구를 포함하지 않습니다.

## 5. 데이터 보관과 삭제

- 앱 설정과 마지막 도시 또는 위치 좌표는 사용자의 기기 내부에 저장됩니다.
- 앱을 삭제하면 기기 내부에 저장된 앱 데이터도 함께 삭제됩니다.
- 운영자가 별도 서버에 사용자 데이터를 저장하지 않습니다.

## 6. 어린이 개인정보 보호

본 앱은 어린이를 주 대상으로 하지 않으며, 어린이의 개인정보를 의도적으로 수집하지 않습니다.

## 7. 변경 고지

앱의 데이터 처리 방식이 바뀌면 이 페이지의 최종 수정일과 내용을 갱신합니다. 중요한 변경이 있는 경우 앱 업데이트 또는 릴리즈 노트를 통해 안내할 수 있습니다.
